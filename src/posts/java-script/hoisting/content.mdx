호이스팅에 대해 알아보기 전에 먼저 이 코드를 보자.

```tsx
console.log(a);
var a = 10;
console.log(a);
```

이 코드를 실행하면 무엇이 출력될까?
a를 선언하기 전에 `console.log(a);`를 실행했으니까 `ReferenceError`?

<br />

결과는 다음과 같다.

```tsx
console.log(a); // undefined
var a = 10;
console.log(a); // 10
```

왜 에러가 안나고 undefined가 출력되는걸까?
이게 바로 호이스팅 때문이다.

---

## 호이스팅이 뭔데?

호이스팅은 변수와 함수 선언이 컴파일 단계에서 맨 위로 끌어올리는 것을 의미한다.
